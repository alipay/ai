# AI-Payment Repo

**Alipay AI Payment Repository** — AI-powered payment product integration methods and development toolkit.

**支付宝 AI 支付仓库** —— AI 赋能的支付产品集成方式与开发工具集。

## 📖 项目定位 | Project Positioning

This repository is an AI-powered payment capability development resource hub containing a series of payment-related AI Agent Skills to help developers and technical teams quickly understand and integrate various payment capabilities from Alipay Open Platform.

本仓库是一个 AI 赋能的支付能力开发资源库，包含一系列支付相关的 AI 智能体 Skills，用于帮助开发者、技术团队快速理解和集成支付宝开放平台的各项支付能力。
