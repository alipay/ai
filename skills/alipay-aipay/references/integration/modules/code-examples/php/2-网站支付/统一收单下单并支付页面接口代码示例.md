# 统一收单下单并支付页面接口 - PHP 示例

## 请求示例

```php
<?php
require_once __DIR__ . '/aop/AopClient.php';
require_once __DIR__ . '/aop/request/AlipayTradePagePayRequest.php';

// 初始化 SDK（PHP 使用 PKCS#1 格式私钥，Java 使用 PKCS#8 格式，请勿混用）
$aop = new AopClient();
$aop->gatewayUrl = 'https://openapi.alipay.com/gateway.do';
// 沙箱环境: $aop->gatewayUrl = 'https://openapi-sandbox.dl.alipaydev.com/gateway.do';
$aop->appId = '<-- 请填写您的AppId，例如：20190xxxxxx -->';
$aop->rsaPrivateKey = '<-- 请填写您的应用私钥（PKCS#1格式），例如：MIIE...（脱敏示例） -->';
$aop->alipayrsaPublicKey = '<-- 请填写您的支付宝公钥，例如：MIIB...（脱敏示例） -->';
$aop->format = 'json';
$aop->postCharset = 'UTF-8';
$aop->signType = 'RSA2';

// 构造请求参数
$request = new AlipayTradePagePayRequest();
$notifyUrl = trim(getenv('ALIPAY_NOTIFY_URL') ?: '');
if ($notifyUrl !== '') {
    $request->setNotifyUrl($notifyUrl);  // 真实生产或公网联调时传入；本地没有公网 HTTPS notify_url 时不要传 null、空字符串或占位 URL
}
$request->setReturnUrl('https://your-domain.com/return');  // 默认传入并替换为已验证结果页；仅用户明确关闭同步回跳时省略

$bizContent = [
    'out_trade_no' => date('YmdHis') . mt_rand(1000, 9999),  // 商户订单号
    'total_amount' => '0.01',                                // 订单总金额
    'subject' => '测试商品',                                   // 订单标题
    'product_code' => 'FAST_INSTANT_TRADE_PAY',              // 网站支付固定值
];
$request->setBizContent(json_encode($bizContent, JSON_UNESCAPED_UNICODE));

// ⚠️ 网站支付必须使用 pageExecute()，不能使用 execute()
// ⚠️ 必须使用 POST 模式：pageExecute($request, 'POST')
$response = $aop->pageExecute($request, 'POST');

// pageExecute 返回的是 HTML 表单字符串，前端需要渲染并自动提交
echo $response;
```

> **⚠️ 关键注意事项：**
> 1. 必须使用 `pageExecute()` 方法，不能使用 `execute()` 方法
> 2. 必须使用 POST 模式：`$aop->pageExecute($request, 'POST')`
> 3. 返回的 HTML 需要前端渲染并自动提交，禁止直接在服务端 redirect
> 4. PHP 使用 PKCS#1 格式私钥，Java 使用 PKCS#8 格式，请勿混用
