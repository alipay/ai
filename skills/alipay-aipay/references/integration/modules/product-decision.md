# 产品决策

用于用户集成支付宝支付产品时的产品决策。**本 Skill 仅支持按量付费、网站支付、APP 支付三种产品**。

---

## 支持的产品

> 术语说明：本 Skill 中“网站支付”是唯一的网页支付产品概念，覆盖电脑网页和手机浏览器网页/H5 场景；“电脑网站支付”“PC网站支付”“手机网站支付”“H5支付”均路由到网站支付。集成侧网站支付底层接口固定使用 `alipay.trade.page.pay`，即使是手机浏览器/H5 场景，也不得改用或推荐其他网页支付接口。

| 产品 | 定义 | 适用场景 |
|------|------|----------|
| **按量付费** | 面向 AI 智能体的机器支付收款产品 | API、数字内容、算力资源、智能体按调用付费、402 协议收款 |
| **网站支付** | 用户在电脑网页或手机网页发起支付 | PC 官网、电商网站、在线教育、政务缴费网站、手机 H5 页面 |
| **APP 支付** | 原生 App 内调起支付宝支付 | iOS/Android/鸿蒙原生 App 内支付 |

---

## 产品决策树

```
用户咨询支付宝接入
        |
        +-- 面向 AI 智能体收款？402 协议？
        |       +-- 按量付费
        |
        +-- 线上网页支付？
        |       +-- 电脑浏览器网页 --> 网站支付
        |       +-- 手机浏览器 H5 --> 网站支付
        |
        +-- 原生 APP 支付？
                +-- iOS/Android/鸿蒙 --> APP 支付
```

## 产品澄清

用户描述模糊或关键词匹配不明确时，执行下列命令固定渲染 `product.clarify`，等待用户以自然语言回复产品名称和业务需求。本模块不维护第二份澄清模板；不得临场增加不支持产品、改写网站支付接口或省略三种支持产品。

```bash
printf '%s' '{}' | node ../../normal/scripts/render_customer_message.mjs product.clarify --variant DEFAULT
```
