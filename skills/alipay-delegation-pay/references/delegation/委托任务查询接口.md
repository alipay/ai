# 委托任务查询接口

## 接口说明

- **接口名称**：`alipay.user.agreement.delegation.query`
- **接口类型**：查询接口
- **请求地址**：`https://openapi.alipay.com/gateway.do`
- **调用方式**：服务端调用 `alipayClient.execute(request)`

该接口用于查询用户的代买委托任务详情，包括任务状态、有效期、剩余次数和金额等信息。

---

## 请求参数

### 业务参数（biz_content）

#### 协议标识（二选一必填）

| 参数 | 类型 | 必填 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| agreement_no | String | 二选一 | 1-32 | 委托协议号 | 20265005004910872660 |
| external_agreement_no | String | 二选一 | 1-32 | 委托外部签约号 | external_agreement_no_001 |

#### 委托标识（二选一必填）

| 参数 | 类型 | 必填 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| delegation_id | String | 二选一 | 1-32 | 代买委托id | 2026001 |
| external_delegation_id | String | 二选一 | 1-32 | 外部代买委托id | exteranl_id_001 |

---

## 响应参数

| 参数 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| status | String | 是 | 代买委托任务当前状态 | NORMAL |
| delegation_id | String | 是 | 委托号 | 20260202002630110000070000000001 |
| external_delegation_id | String | 是 | 外部委托号 | external_delegation_id_001 |
| delegation_tag | String | 是 | 代买委托场景 | 购买车票 |
| validity_start_time | String | 是 | 有效期开始时间（yyyy-MM-dd HH:mm:ss） | 2026-01-29 00:00:00 |
| validity_end_time | String | 是 | 有效期结束时间（yyyy-MM-dd HH:mm:ss） | 2026-01-31 00:00:00 |
| max_total_amount | String | 是 | 总金额上限（元） | 10 |
| remaining_amount | String | 是 | 剩余可用金额（元） | 10 |
| times_limit | String | 是 | 总次数上限 | 5 |
| remaining_times | String | 是 | 剩余次数 | 5 |

## 错误码

| 错误码 | 错误描述 | 解决方案 |
|--------|---------|---------|
| SYSTEM_ERROR | 系统繁忙 | 服务器异常，可能发生了网络或系统异常，商户可以用同样的请求发起重试 |
| INVALID_PARAMETER | 参数有误 | 请根据接口返回的具体错误信息，修改参数后进行重试 |
| DELEGATION_NOT_EXIST | 代买委托任务不存在 | 可能是传入了错误的代买委托号，或外部代买委托号关联的任务没有签约成功 |
| USER_AGREEMENT_NOT_EXIST | 用户协议不存在 | 可能是传入错误的用户协议号，或协议已经被解约 |

## 请求示例

```java
// 初始化SDK
AlipayClient alipayClient = new DefaultAlipayClient(getAlipayConfig());

// 构造请求
AlipayUserAgreementDelegationQueryRequest request = new AlipayUserAgreementDelegationQueryRequest();
AlipayUserAgreementDelegationQueryModel model = new AlipayUserAgreementDelegationQueryModel();

// 设置协议标识（二选一）
model.setAgreementNo("20265005004910872660");
// 或者使用外部签约号
// model.setExternalAgreementNo("external_agreement_no_001");

// 设置委托标识（二选一）
model.setDelegationId("2026001");
// 或者使用外部委托号
// model.setExternalDelegationId("exteranl_id_001");

request.setBizModel(model);

// 执行请求
AlipayUserAgreementDelegationQueryResponse response = alipayClient.execute(request);

if (response.isSuccess()) {
    // 查询成功，获取委托任务信息
    String status = response.getStatus();
    String remainingAmount = response.getRemainingAmount();
    String remainingTimes = response.getRemainingTimes();
    System.out.println("委托状态: " + status);
    System.out.println("剩余金额: " + remainingAmount);
    System.out.println("剩余次数: " + remainingTimes);
} else {
    // 查询失败
    System.out.println("错误码: " + response.getSubCode());
    System.out.println("错误信息: " + response.getSubMsg());
}
```
