# 委托代买申请接口

## 接口说明

- **接口名称**：`alipay.user.agreement.delegation.apply`
- **接口类型**：签名数据准备接口
- **请求地址**：`https://openapi.alipay.com/gateway.do`
- **调用方式**：服务端调用 `alipayClient.sdkExecute(request)`，获取 `response.getBody()` 作为签名字符串

该接口用于生成可信签名字符串（orderStr），包含业务参数及商户身份信息，用于唤起支付宝客户端完成用户委托代买授权签约。

---

## 请求参数

### 业务参数（biz_content）

| 参数 | 类型 | 必填 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| agent_id | String | 是 | 1-32 | 智能体id | agent_id_001 |
| external_agreement_no | String | 是 | 1-32 | 商户签约号，需在商户系统中唯一，支持大小写字母和数字 | agreementno001 |
| personal_product_code | String | 是 | 1-32 | 个人签约产品码，固定值 | UAM_AGENT_AUTH_P |
| delegation_params | DelegationParams | 是 | - | 委托代买信息 | 见下表 |
| access_params | AccessParams | 是 | - | 接入渠道参数 | 见下表 |
| conversation_history | Conversation[] | 是 | 0-10 | 对话历史 | 见下表 |

### DelegationParams 结构

| 参数 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| external_delegation_id | String | 是 | 商户委托单号 | external_delegation_0001 |
| delegation_desc | String | 是 | 委托任务描述 | 监控1月20日北京去哈尔滨的高铁二等座 |
| delegation_tag | String | 否 | 委托标签 | 购买火车票 |
| max_total_amount | String | 是 | 最大授权金额（元） | 10.86 |
| times_limit | String | 否 | 最大执行次数 | 5 |
| validity_start_time | String | 是 | 生效开始时间（yyyy-MM-dd） | 2026-01-30 |
| validity_end_time | String | 是 | 生效结束时间（yyyy-MM-dd） | 2026-02-22 |

### AccessParams 结构

| 参数 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| channel | String | 是 | 接入渠道，固定值 | ALIPAYAPP |

### Conversation 结构

| 参数 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| role | String | 是 | 角色：USER 或 ASSISTANT | USER |
| content | String | 是 | 对话内容 | 帮我抢一张1月20号回哈尔滨的票，二等座。 |
| create_time | String | 是 | 对话时间（yyyy-MM-dd HH:mm:ss） | 2025-12-23 10:28:00 |

> ⚠️ 不同 SDK 版本对该字段的序列化可能存在差异（下划线 `create_time` / 驼峰 `createTime`）。若调用成功但会话历史未在支付宝客户端展示，可切换命名再试，详见 [故障排查 §2](../故障排查.md#2-委托代买申请会话历史不生效)。

## 响应参数

该接口通过 `sdkExecute` 调用，响应通过 `response.getBody()` 获取。

| 参数 | 类型 | 说明 |
|------|------|------|
| orderStr | String | 签名后的业务数据字符串，最大长度 16384，用于唤起支付宝客户端 |

### 响应示例

```
app_id=2017060101317939&biz_content=%7B%22agent_id%22%3A%22agent_id_001%22%2C%22personal_product_code%22%3A%22UAM_AGENT_AUTH_P%22%2C%22external_agreement_no%22%3A%22agreementno001%22%7D&charset=UTF-8&format=json&method=alipay.user.agreement.delegation.apply&sign=ERITJKEIJKJHKKKKKKKHJEREEEEEEEEEEE&sign_type=RSA2&timestamp=2014-07-24+03%3A07%3A50&version=1.0
```

## 请求示例

```java
// 初始化SDK
AlipayClient alipayClient = new DefaultAlipayClient(getAlipayConfig());

// 构造请求
AlipayUserAgreementDelegationApplyRequest request = new AlipayUserAgreementDelegationApplyRequest();
AlipayUserAgreementDelegationApplyModel model = new AlipayUserAgreementDelegationApplyModel();

// 设置智能体id
model.setAgentId("agent_id_001");

// 设置商户签约号
model.setExternalAgreementNo("agreementno001");

// 设置个人签约产品码
model.setPersonalProductCode("UAM_AGENT_AUTH_P");

// 设置对话历史
List<Conversation> conversationHistory = new ArrayList<>();
Conversation conversation = new Conversation();
conversation.setRole("USER");
conversation.setContent("帮我抢一张1月20号回哈尔滨的票，二等座。");
conversation.setCreateTime("2025-12-23 10:28:00");
conversationHistory.add(conversation);
model.setConversationHistory(conversationHistory);

// 设置委托参数
DelegationParams delegationParams = new DelegationParams();
delegationParams.setExternalDelegationId("external_delegation_0001");
delegationParams.setDelegationDesc("监控1月20日北京去哈尔滨的高铁二等座");
delegationParams.setDelegationTag("购买火车票");
delegationParams.setMaxTotalAmount("10.86");
delegationParams.setTimesLimit("5");
delegationParams.setValidityStartTime("2026-01-30");
delegationParams.setValidityEndTime("2026-02-22");
model.setDelegationParams(delegationParams);

// 设置接入渠道
AccessParams accessParams = new AccessParams();
accessParams.setChannel("ALIPAYAPP");
model.setAccessParams(accessParams);

request.setBizModel(model);

// 执行请求，获取签名字符串
AlipayUserAgreementDelegationApplyResponse response = alipayClient.sdkExecute(request);
String orderStr = response.getBody();
```
