# 预下单接口（统一收单交易订单预支付）

## 接口说明

- **接口名称**：`alipay.trade.order.prepay`
- **接口类型**：商户服务端接口（标准收单接口）
- **请求地址**：`https://openapi.alipay.com/gateway.do`
- **调用方式**：商户服务端调用 `alipayClient.execute(request)`
- **调用主体**：**商户**（注意：不是智能体平台。商户用订单信息换取 `prepay_id`，再交给智能体发起委托支付）

该接口用于在委托支付/AI付扣款**之前**生成预下单 ID（`prepay_id`）。委托支付接口（`alipay.trade.agent.delegation.pay`）必须先拿到 `prepay_id` 才能发起扣款。

> **前置权限**：商户需先开通 **App 支付** 产品权限（联系支付宝业务人员开通）。未开通会返回 `ACCESS_FORBIDDEN`（40006 接口调用权限不足）。

---

## 智能体代理支付场景的关键配置

在智能体委托支付场景下，以下 3 个参数必须按规定填写，否则**不会返回 `prepay_id`**：

| 参数 | 取值 | 说明 |
|------|------|------|
| `payment_type` | `agent_pay` | **必填**。智能体代理支付方式，不传则不返回 prepay_id |
| `product_code` | `QUICK_MSECURITY_PAY` | 智能体场景固定值 |
| `payment_inst` | （不传） | 智能体场景无需传递 |

---

## 请求参数

### 业务参数（biz_content）— 核心字段

| 参数 | 必填 | 类型 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| total_amount | 是 | Price | 11 | 订单总金额（元），范围 [0.01, 100000000] | 88.88 |
| subject | 是 | String | 256 | 订单标题，不可含特殊字符（`/ = &` 等） | 早餐盲盒 |
| out_trade_no | 是 | String | 64 | 商户订单号，商户端唯一，字母数字下划线 | 20150320010101001 |
| product_code | 否 | String | 64 | 产品码，智能体场景固定 `QUICK_MSECURITY_PAY` | QUICK_MSECURITY_PAY |
| payment_type | 否 | String | 64 | 智能体场景**必填** `agent_pay` | agent_pay |
| seller_id | 否 | String | 28 | 卖家支付宝用户 ID，空则默认商户签约账号 | 2088102146225135 |
| body | 否 | String | 128 | 订单附加信息，异步通知/对账单原样返回 | 早餐盲盒套餐 |
| time_expire / timeout_express | 否 | — | — | 绝对/相对超时时间，二选一（都传以 time_expire 为准） | 90m |
| merchant_return_url | 否 | String | 1024 | 支付成功回跳商家 APP 地址 | boostapp://... |

> 完整字段（分账 `royalty_info`、结算 `settle_info`、商品明细 `goods_detail` 等）见支付宝开放平台原文。

## 响应参数

| 参数 | 必返回 | 类型 | 说明 |
|------|--------|------|------|
| out_trade_no | 是 | String(64) | 商户订单号 |
| trade_no | 否 | String(64) | 支付宝交易号 |
| tn | 否 | String(256) | 账户机构唤端需要的订单唯一标识 |
| **prepay_id** | 否 | String(512) | **预下单 ID，用于后续委托支付（默认有效期 2 小时）** |
| jump_info | 否 | JumpInfo | 支付跳转信息（normal_url / applink_url / scheme_url / app_identifier） |

## 错误码

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| SYSTEM_ERROR | 系统繁忙 | 用同样请求重试 |
| INVALID_PARAMETER | 参数有误 | 修改参数后重试 |
| ACCESS_FORBIDDEN | 无权限使用接口 | 智能体场景需先开通 **App 支付** 产品（40006） |
| BUYER_ENABLE_STATUS_FORBID | 买家状态非法 | 用户联系支付宝确认 |
| BUYER_SELLER_EQUAL | 买卖家不能相同 | 更换买家 |
| CONTEXT_INCONSISTENT | 交易信息被篡改 | 参数与已存交易不一致，更换订单号重发 |
| EXIST_FORBIDDEN_WORD | 含违禁词 | 修改订单信息 |
| NO_PAYMENT_INSTRUMENTS_AVAILABLE | 无可用支付工具 | 更换付款方式 |
| PARTNER_ERROR | APP_ID 填写错误 | 联系支付宝确认 APP_ID 状态 |
| TOTAL_FEE_EXCEED | 订单总金额超范围 | 修改金额 |
| TRADE_HAS_CLOSE | 交易已关闭 | 更换订单号重发 |
| TRADE_HAS_SUCCESS | 交易已被支付 | 确认是否已成功 |

## 请求示例

### cURL

```bash
curl 'https://openapi.alipay.com/gateway.do?charset=UTF-8&method=alipay.trade.order.prepay&format=json&sign=${sign}&app_id=${appid}&version=1.0&sign_type=RSA2&timestamp=${now}' \
 -F 'app_auth_token=${app_auth_token}' \
 -F 'biz_content={
    "total_amount":"88.88",
    "product_code":"QUICK_MSECURITY_PAY",
    "subject":"早餐盲盒",
    "out_trade_no":"20150320010101001",
    "payment_type":"agent_pay"
}'
```

### Java

```java
// 初始化SDK
AlipayClient alipayClient = new DefaultAlipayClient(getAlipayConfig());

// 构造请求
AlipayTradeOrderPrepayRequest request = new AlipayTradeOrderPrepayRequest();
AlipayTradeOrderPrepayModel model = new AlipayTradeOrderPrepayModel();

// 必填：订单信息
model.setOutTradeNo("20150320010101001");
model.setTotalAmount("88.88");
model.setSubject("早餐盲盒");

// 智能体代理支付场景的关键配置
model.setProductCode("QUICK_MSECURITY_PAY");
model.setPaymentType("agent_pay");

request.setBizModel(model);

// 执行请求
AlipayTradeOrderPrepayResponse response = alipayClient.execute(request);

if (response.isSuccess()) {
    String prepayId = response.getPrepayId();   // 交给智能体用于委托支付
    System.out.println("预下单成功，prepay_id: " + prepayId);
    // TODO: 记录 prepay_id（有效期 2 小时），返回给智能体发起 alipay.trade.agent.delegation.pay
} else {
    System.out.println("错误码: " + response.getSubCode());
    System.out.println("错误信息: " + response.getSubMsg());
}
```

## 在委托支付链路中的位置

```
查询委托状态(status=NORMAL)
   → 商户预下单 alipay.trade.order.prepay  → prepay_id（本接口）
   → 委托支付 alipay.trade.agent.delegation.pay(prepay_id + agreement_no + delegation_id) → trade_no
```
