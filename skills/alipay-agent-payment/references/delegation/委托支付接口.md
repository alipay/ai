# 委托支付接口

## 接口说明

- **接口名称**：`alipay.trade.agent.delegation.pay`
- **接口类型**：支付接口
- **请求地址**：`https://openapi.alipay.com/gateway.do`
- **调用方式**：服务端调用 `alipayClient.execute(request)`

该接口用于智能体平台基于委托ID对商户订单进行扣款。需先通过 `alipay.trade.order.prepay` 接口获取预下单ID（详见 [预下单接口.md](预下单接口.md)）。

---

## 请求参数

### 业务参数（biz_content）

| 参数 | 类型 | 必填 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| prepay_id | String | 是 | 1-512 | 预下单ID，通过 alipay.trade.order.prepay 接口获取 | MV91dHAucHJlb3JkZXJfMjAyNTA2MDRf... |
| agreement_no | String | 是 | 1-128 | Agent签约协议号，用户签约成功后的协议号 | 20170322450983769228 |
| delegation_id | String | 是 | 1-64 | 代买委托id | 2026001123456789 |

---

## 响应参数

| 参数 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| trade_no | String | 是 | 支付宝交易号 | 2013112011001004330000121536 |

## 请求示例

```java
// 初始化SDK
AlipayClient alipayClient = new DefaultAlipayClient(getAlipayConfig());

// 构造请求
AlipayTradeAgentDelegationPayRequest request = new AlipayTradeAgentDelegationPayRequest();
AlipayTradeAgentDelegationPayModel model = new AlipayTradeAgentDelegationPayModel();

// 设置预下单ID（通过 alipay.trade.order.prepay 接口获取）
model.setPrepayId("MV91dHAucHJlb3JkZXJfMjAyNTA2MDRfMjA4ODg0MTAzMzUzMTQ0OF9YUDE1MjUwNjA0MDU1MDA2MDQzNTAyOTQwMDY5OTJfMzMwNzg2XzQyLTIwNzYxJDgq");

// 设置Agent签约协议号
model.setAgreementNo("20170322450983769228");

// 设置代买委托id
model.setDelegationId("2026001123456789");

request.setBizModel(model);

// 执行请求
AlipayTradeAgentDelegationPayResponse response = alipayClient.execute(request);

if (response.isSuccess()) {
    // 支付成功
    String tradeNo = response.getTradeNo();
    System.out.println("支付成功，支付宝交易号: " + tradeNo);
} else {
    // 支付失败
    System.out.println("错误码: " + response.getSubCode());
    System.out.println("错误信息: " + response.getSubMsg());
}
```
