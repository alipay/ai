# 委托任务取消接口

## 接口说明

- **接口名称**：`alipay.user.agreement.delegation.cancel`
- **接口类型**：操作接口
- **请求地址**：`https://openapi.alipay.com/gateway.do`
- **调用方式**：服务端调用 `alipayClient.execute(request)`

该接口用于主动取消进行中的代买委托任务。

---

## 请求参数

### 业务参数（biz_content）

| 参数 | 类型 | 必填 | 长度 | 说明 | 示例值 |
|------|------|------|------|------|--------|
| agreement_no | String | 是 | 1-32 | 委托协议号 | 20265002005167619007 |
| delegation_id | String | 是 | 1-32 | 代买委托号 | 20260202002630110000070000000001 |

---

## 响应参数

该接口无业务响应参数，仅返回公共响应参数。

| 参数 | 类型 | 说明 |
|------|------|------|
| code | String | 结果码，10000 表示成功 |
| msg | String | 结果描述 |

### 响应示例

```json
{
  "alipay_user_agreement_delegation_cancel_response": {
    "code": "10000",
    "msg": "Success"
  },
  "sign": "ERITJKEIJKJHKKKKKKKHJEREEEEEEEEEEE"
}
```

---

## 错误码

| 错误码 | 错误描述 | 解决方案 |
|--------|---------|---------|
| SYSTEM_ERROR | 系统繁忙 | 服务器异常，可能发生了网络或系统异常，商户可以用同样的请求发起重试 |
| INVALID_PARAMETER | 参数有误 | 请根据接口返回的具体错误信息，修改参数后进行重试 |

---

## 请求示例

```java
// 初始化SDK
AlipayClient alipayClient = new DefaultAlipayClient(getAlipayConfig());

// 构造请求
        AlipayUserAgreementDelegationCancelRequest request = new AlipayUserAgreementDelegationCancelRequest();
        AlipayUserAgreementDelegationCancelModel model = new AlipayUserAgreementDelegationCancelModel();

// 设置委托协议号
        model.setAgreementNo("20265002005167619007");

// 设置代买委托号
        model.setDelegationId("20260202002630110000070000000001");

        request.setBizModel(model);

// 执行请求
        AlipayUserAgreementDelegationCancelResponse response = alipayClient.execute(request);

        if (response.isSuccess()) {
        // 取消成功
        System.out.println("委托任务取消成功");
        } else {
        // 取消失败
        System.out.println("错误码: " + response.getSubCode());
        System.out.println("错误信息: " + response.getSubMsg());
        }
```
